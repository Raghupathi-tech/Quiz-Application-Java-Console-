# Quiz Application

## Description
A console-based Quiz Application developed in Java. Users answer multiple-choice questions and receive a final score at the end.

## Features
- Multiple-choice questions
- Score calculation
- User input validation
- Easy-to-use interface

## Technologies Used
- Java
- Scanner Class

## How to Run
1. Download the project.
2. Open in Eclipse or IntelliJ IDEA.
3. Run `QuizApplication.java`.
